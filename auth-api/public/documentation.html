API Documentation

Endpoints

Register

POST /register

Registers a new user.

  {
    "username": "string",
    "email": "string",
    "password": "string"
  }

Login

POST /login

Logs in a user and returns a JWT token.

  {
    "username": "string",
    "password": "string"
  }

Verify Token

GET /verify-token

Verifies the provided JWT token.

  Headers: {
    "Authorization": "Bearer <token>"
  }

Current User

GET /current-user

Returns the current authenticated user.

  Headers: {
    "Authorization": "Bearer <token>"
  }

Active Tokens

GET /active-tokens

Returns the list of active tokens.

  Headers: {
    "Authorization": "Bearer <token>"
  }

Forgot Password

POST /forgot-password

Sends a password reset link to the provided email.

  {
    "email": "string"
  }

Reset Password

POST /reset-password

Resets the password using the provided token and new password.

  {
    "token": "string",
    "newPassword": "string"
  }

Logout

POST /logout

Logs out the current user.

  Headers: {
    "Authorization": "Bearer <token>"
  }
